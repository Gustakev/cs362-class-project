<h1 align="center">iExtract Installation</h1>


## Prerequisites
- Python 3.13 or newer
- Git

## Installation
1. Open your terminal and clone the repo with the command:
```bash
git clone https://github.com/Gustakev/cs362-class-project.git
```
2. Move into the repo with the command:
```bash
cd cs362-class-project
```

3. Start a virtual environment with the command:
```bash
# On Windows
python -m venv venv
venv/Scripts/activate

# On Mac/Linux
python3 -m venv venv
source venv/bin/activate
```

4. Install the required dependencies with the command:
```bash
pip install -r requirements.txt
```

5.  Start the program with the command:
```bash
python iExtract.py
```

## Verify Installation

If the installation was successful, running the program should display the main menu in the terminal.
